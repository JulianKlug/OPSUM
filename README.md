# OPSUM
Outcome Prediction for Stroke Unit high-frequency data through Machine learning 

### Investigated outcomes
1. Short term
- early neurological deterioration


2. Medium and long-term
- Mortality: '3M Death' & 'Death in hospital'
- Functional outcome: '3M mRS 0-2'

### Model weights
1. Short term: https://doi.org/10.5281/zenodo.19596386 
2. Long term: https://doi.org/10.5281/zenodo.8195710
